## Installation & Setup

### 1 Clone the repository

```bash
git clone https://github.com/rupeshkumargupta5532/Artiset-Job-Portal-Frontend-Developer-.git
cd Artiset-Job-Portal-Frontend-Developer-
npm install
npm run dev
```
